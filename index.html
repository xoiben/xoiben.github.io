Privacy & Technical Note

1. General Information
This website is a strictly private, non-commercial hobby project. The operator does not collect, store, or process any personal data.

2. No Tracking & Client-Side Decryption
This website does not use tracking tools or cookies. Decryption happens entirely within your browser using a shared password; no individual user accounts are created, and the password you enter is never transmitted to a server. If you choose to stay logged in, an encrypted cryptographic token is saved locally in your browser's local storage so you do not have to re-enter the password. No data is ever transmitted to us.

3. Hosting via GitHub Pages
This page is hosted via "GitHub Pages" (GitHub Inc., USA). When accessing it, GitHub automatically collects server log files, including your IP address. This is technically necessary to safely deliver the static files to your browser (Art. 6(1)(f) GDPR). Data transfer to the USA is safeguarded by GitHub's Standard Contractual Clauses. For more details, review the official GitHub Privacy Statement.
